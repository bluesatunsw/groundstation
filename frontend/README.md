# Front-end

The frontend code to control the Bluesat Autonomous Groundstation.

## Development

*Installation*

`npm install`

`npm run build`

*Testing*

`npm run lint-ts`

`npm run lint-html`

`npm run lint-css`

*Formatting*

`npm run fix-ts`

`npm run fix-css`

## Hosting

`python serve.py`

The frontend can be accessed from localhost:5001.
